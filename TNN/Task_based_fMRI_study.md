Task based fMRI vs resting state fMRI

Task based fMRI: task에 따라 특정 뇌 영역을 활선화시켜서 특정 활동이나 작업을 수행할 때 뇌 활동을 관찰하는 기술.

```
Experiment Design
  Block Design: rest를 주기적으로 반복
  Event-related Design: jitter 구간을 통해 신호가 중첩되는 것을 방지함
```
